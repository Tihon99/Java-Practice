Методы класса Arrays:

1. toString() - преобразует массив в строку, удобную для чтения

2. binarySearch() - выполняет бинарный поиск элемента в отсортированном массиве
   - Возвращает индекс элемента, если он найден
   - Возвращает отрицательное число, если элемент не найден
   - Массив должен быть отсортирован перед использованием метода

3. equals() - сравнивает два массива на равенство
   - Возвращает true, если массивы имеют одинаковую длину и содержат одинаковые элементы
   - Порядок элементов важен

4. compare() - сравнивает два массива лексикографически
   - Возвращает отрицательное число, если первый массив меньше второго
   - Возвращает положительное число, если первый массив больше второго
   - Возвращает 0, если массивы равны

5. sort() - сортирует массив по возрастанию
   - Изменяет исходный массив
   - Использует модифицированную быструю сортировку для примитивных типов
