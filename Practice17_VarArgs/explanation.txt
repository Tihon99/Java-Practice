Практика 17: Методы с переменным числом параметров

Класс A содержит три перегруженных метода print с переменным числом параметров:

1. print(int... numbers) - принимает любое количество целых чисел
2. print(double... numbers) - принимает любое количество чисел с плавающей точкой
3. print(String first, int... numbers) - принимает строку и любое количество целых чисел

Вывод программы:
int args: 1 2 3
double args: 1.1 2.2 3.3
Numbers: 1 2 3
